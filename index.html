Dados abertos: Orçamento Secreto

O trabalho desenvolvido nas matérias de Projeto Integrador I e II é sobre o "Orçamento Secreto" que trata sobre as Ementas do relator. O objetivo é mostrar a população de forma transparente as informações da emenda.

Segundo semestre

No segundo semestre, cada um do grupo exerceu uma
diferente função. Como pode ser visto abaixo

  • Adam Documentação
  • Gabriela Front-end
  • Gustavo Back-end
  • Matheus Filtragem de dados
  • Willian Servidor

Tive a oportunidade de documentar um site que mostra dados das emendas do relator. Foi uma experiência incrível, pois pude aumentar a transparência e permitir que as pessoas entendam melhor as decisões legislativas. Pesquisei, organizei as informações e verifiquei os dados para garantir sua precisão. O feedback positivo dos usuários mostrou como o acesso a informações transparentes fortalece a democracia. Foi gratificante contribuir para uma participação cidadã mais informada.
